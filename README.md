# 🍕 Pizza Sales Data Analysis (SQL)

## 📌 Project Overview

This project analyzes pizza sales data using SQL to find insights like sales performance, popular pizzas, and customer ordering trends.

## 🛠 Tools Used

* MySQL
* SQL

## 📂 Dataset

This project uses 4 datasets:

* orders.csv
* order_details.csv
* pizzas.csv
* pizza_types.csv

## 🔍 What I Did

* Calculated total orders and revenue
* Found most popular pizza size
* Identified top-selling pizzas
* Analyzed order trends by time
* Calculated revenue and category performance

## 📊 Key Insights

* Some pizzas generate more revenue than others
* Orders are higher during specific hours
* Certain categories are more popular
